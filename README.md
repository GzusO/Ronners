# Ronners 

### Ronners The Discord Bot

## Built With

* [Discord.Net (Version - 3.12.0)](https://github.com/discord-net/Discord.Net) - The Discord Library.
* [ImageSharp (Version - 2.1.3)](https://github.com/SixLabors/ImageSharp) - Image Processing Library.

## Building

1. ``dotnet restore``
